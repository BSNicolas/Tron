# Tron
Jeu Tron en java réalisé à l'université en 2023 pour un projet de programmation

# Lancer le jeu
- Dans un terminal à la racine du projet, tapez cette commande : `javac *`
- Ensuite, dans le même terminal toujours à la racine du projet, tapez cette commande : `java Creation`
- Une interface se lance vous demandant de définir les paramètres de la partie

# Fonctionnement du jeu
- Le Joueur 1 se trouve en haut à gauche et le Joueur 2 en bas à droite
- Des obstacles sont générés aléatoirement sur le terrain
- L'objectif est de bloquer l'adversaire dans une plus petite zone que la notre
- Le perdant est le premier qui n'a plus aucune case de disponible

# Problèmes
- Plus la partie dure longtemps et plus le terrain est grand, plus il y aura des problèmes de performances et des ralentissments
- L'interface de jeu est toute petite
